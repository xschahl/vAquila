title: Public Beta Notes
---

vAquila is currently in public beta.

## What is stable

- CLI and Docker orchestration flow
- GPU and CPU launch paths
- Basic Web UI operations

## What is evolving

- VRAM estimation precision across wider model/runtime combinations
- Startup tuning heuristics under fragmented memory conditions

## Feedback expected

When reporting an issue, include:

- OS + GPU details
- exact command/options
- model id
- relevant container logs
